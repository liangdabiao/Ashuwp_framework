# Ashuwp_framework
##简介

Ashuwp_framework是由阿树工作室发布的一个wordpress后台框架。

使用本框架的要求：你需要具备基础的php语法知识，以及对wordpress主题有基本的了解。

##本框架的功能

* 给wordpress文章添加自定义字段
    * 支持自定义文章类型
* 给wordpress主题后台添加设置页面
    * 支持多个顶级页面及子页面
    * 支持设置选项的导入导出
* 给wordpress的分类添加自定义字段
    * 支持tag等自定义分类法
