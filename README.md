# Ashuwp_framework
##简介

Ashuwp_framework是由阿树工作室发布的一个wordpress后台框架。

最新版本 3.1
更新日期 2015.9.30

使用本框架的要求：你需要具备基础的php语法知识，以及对wordpress主题有基本的了解。

##本框架的功能

* 给wordpress文章添加自定义字段
    * 支持自定义文章类型
* 给wordpress主题后台添加设置页面
    * 支持多个顶级页面及子页面
    * 支持设置选项的导入导出
* 给wordpress的分类添加自定义字段
    * 支持tag等自定义分类法
    
##本框架支持的表单类型

以下所有表单项同时支持文章自定义字段、分类自定义字段、设置页面

* 普通提示文字
* 文本框 text
* 文本域 textarea
* 文件上传 使用media upload
* 单选框 radio
* 复选框 checkbox
* 下拉框 select
* 文本编辑器 tinyMCE
* 相册 gallery

##其它

* 文章自定义字段支持选项卡样式
* 设置页面支持选项卡样式
* 分类字段不支持选项卡样式
